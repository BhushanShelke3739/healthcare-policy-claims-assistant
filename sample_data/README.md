# sample_data/

Synthetic fixtures used by the project. **Nothing here is real.** Do not
add real patient records, real claims, real PHI, or real PII under any
circumstances.

| Folder            | What goes here                                            | Filled in by |
|-------------------|-----------------------------------------------------------|--------------|
| `policies/`       | Synthetic policy documents (prior auth, appeals, ...)     | Phase 2      |
| `claims/`         | Synthetic claim summaries (denial reasons, line items)    | Phase 5      |
| `eval_questions/` | Question/answer pairs used by the evaluation harness      | Phase 6      |

Each subfolder has its own README describing the file format expected
there.
